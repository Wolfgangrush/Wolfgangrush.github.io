## ఇన్‌స్టాలేషన్

1. ఏదైనా repo కి వెళ్లండి → Releases → తాజా `.mcpb` ఫైల్‌ను డౌన్‌లోడ్ చేయండి
2. Claude Desktop App తెరవండి → Settings (గేర్ ఐకాన్) → **Extensions** → Install Extension
3. డౌన్‌లోడ్ చేసిన `.mcpb` ఫైల్‌ను ఎంచుకోండి → Open

(గమనిక: Settings → **Extensions** సరైన ప్యానెల్. Plugins లేదా Connectors కాదు.)

## ⚠️ ధృవీకరణ నిరాకరణ

AI తప్పులు చేయగలదు. ప్రతి ముసాయిదా ఒక **ప్రారంభ స్థానం**. దాఖలు చేయడానికి ముందు, ప్రతి శాసన సూచన, ఉల్లేఖనం, వాస్తవ హక్కు, రిజిస్ట్రీ-ఫార్మాట్ ధృవీకరించండి. దాఖలు చేసే న్యాయవాది విషయానికి బాధ్యత వహిస్తాడు.

## 🔒 Pseudonymisation Procedure

Reader ఏజెంట్ డౌన్‌స్ట్రీమ్ LLM-నడిచే ఏజెంట్లు వాస్తవాలను చూడకముందే — పార్టీల పేర్లు, చిరునామాలు, గుర్తింపు సంఖ్యలు, ఆర్థిక సంఖ్యలు — నిర్మాణ placeholder లతో భర్తీ చేస్తుంది. నిజమైన డేటా ఎప్పుడూ LLM కి వెళ్లదు.
