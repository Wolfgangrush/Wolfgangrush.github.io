## ઇન્સ્ટોલેશન

1. કોઈપણ repo પર જાઓ → Releases → નવીનતમ `.mcpb` ફાઇલ ડાઉનલોડ કરો
2. Claude Desktop App ખોલો → Settings (ગિયર આઇકન) → **Extensions** → Install Extension
3. ડાઉનલોડ કરેલી `.mcpb` ફાઇલ પસંદ કરો → Open

(નોંધ: Settings → **Extensions** સાચું પેનલ છે. Plugins કે Connectors નહીં.)

## ⚠️ ચકાસણી અસ્વીકરણ

AI ભૂલો કરી શકે છે. દરેક ડ્રાફ્ટ એક **પ્રારંભિક બિંદુ** છે. દાખલ કરતા પહેલા દરેક કાનૂની સંદર્ભ, ઉલ્લેખ, હકીકત-દાવો અને રજિસ્ટ્રી-ફોર્મેટ ચકાસો. દાખલ કરનાર વકીલ સામગ્રી માટે જવાબદાર રહે છે.

## 🔒 Pseudonymisation Procedure

Reader એજન્ટ ડાઉનસ્ટ્રીમ LLM-સંચાલિત એજન્ટ્સ હકીકતો જુએ તે પહેલા — પક્ષોના નામ, સરનામા, ઓળખ સંખ્યાઓ, નાણાકીય આંકડા — સંરચનાત્મક placeholder થી બદલે છે. વાસ્તવિક ડેટા ક્યારેય LLM માં જતો નથી.
