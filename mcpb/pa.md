## ਸਥਾਪਨਾ

1. ਕਿਸੇ ਵੀ repo 'ਤੇ ਜਾਓ → Releases → ਨਵੀਨਤਮ `.mcpb` ਫਾਈਲ ਡਾਊਨਲੋਡ ਕਰੋ
2. Claude Desktop App ਖੋਲ੍ਹੋ → Settings (ਗੀਅਰ ਆਈਕਾਨ) → **Extensions** → Install Extension
3. ਡਾਊਨਲੋਡ ਕੀਤੀ `.mcpb` ਫਾਈਲ ਚੁਣੋ → Open

(ਨੋਟ: Settings → **Extensions** ਸਹੀ ਪੈਨਲ ਹੈ। Plugins ਜਾਂ Connectors ਨਹੀਂ।)

## ⚠️ ਤਸਦੀਕ ਬੇਦਾਅਵਾ

AI ਗਲਤੀਆਂ ਕਰ ਸਕਦਾ ਹੈ। ਹਰ ਖਰੜਾ ਇੱਕ **ਸ਼ੁਰੂਆਤੀ ਬਿੰਦੂ** ਹੈ। ਦਾਖਲ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਹਰ ਕਾਨੂੰਨੀ ਹਵਾਲਾ, ਉਦਾਹਰਣ, ਤੱਥ-ਦਾਅਵਾ ਅਤੇ ਰਜਿਸਟਰੀ-ਫਾਰਮੈਟ ਦੀ ਤਸਦੀਕ ਕਰੋ। ਦਾਖਲ ਕਰਨ ਵਾਲਾ ਵਕੀਲ ਸਮੱਗਰੀ ਲਈ ਜ਼ਿੰਮੇਵਾਰ ਰਹਿੰਦਾ ਹੈ।

## 🔒 Pseudonymisation Procedure

Reader ਏਜੰਟ ਡਾਊਨਸਟ੍ਰੀਮ LLM-ਚਲਾਏ ਏਜੰਟ ਤੱਥ ਦੇਖਣ ਤੋਂ ਪਹਿਲਾਂ — ਪਾਰਟੀਆਂ ਦੇ ਨਾਮ, ਪਤੇ, ਪਛਾਣ ਨੰਬਰ, ਵਿੱਤੀ ਅੰਕੜੇ — ਢਾਂਚਾਗਤ placeholder ਨਾਲ ਬਦਲਦਾ ਹੈ। ਅਸਲ ਡਾਟਾ ਕਦੇ LLM ਵਿੱਚ ਨਹੀਂ ਜਾਂਦਾ।
