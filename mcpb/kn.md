## ಸ್ಥಾಪನೆ

1. ಯಾವುದೇ repo ಗೆ ಹೋಗಿ → Releases → ಇತ್ತೀಚಿನ `.mcpb` ಫೈಲ್ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ
2. Claude Desktop App ತೆರೆಯಿರಿ → Settings (ಗೇರ್ ಐಕಾನ್) → **Extensions** → Install Extension
3. ಡೌನ್‌ಲೋಡ್ ಮಾಡಿದ `.mcpb` ಫೈಲ್ ಆಯ್ಕೆಮಾಡಿ → Open

(ಗಮನಿಸಿ: Settings → **Extensions** ಸರಿಯಾದ ಪ್ಯಾನೆಲ್. Plugins ಅಥವಾ Connectors ಅಲ್ಲ.)

## ⚠️ ಪರಿಶೀಲನೆ ನಿರಾಕರಣೆ

AI ತಪ್ಪು ಮಾಡಬಹುದು. ಪ್ರತಿ ಕರಡು ಒಂದು **ಆರಂಭಿಕ ಬಿಂದು**. ಸಲ್ಲಿಸುವ ಮೊದಲು ಪ್ರತಿ ಶಾಸನಬದ್ಧ ಉಲ್ಲೇಖ, ಉಲ್ಲೇಖ, ವಾಸ್ತವಿಕ-ಹಕ್ಕು ಮತ್ತು ರಿಜಿಸ್ಟ್ರಿ-ಸ್ವರೂಪವನ್ನು ಪರಿಶೀಲಿಸಿ. ಸಲ್ಲಿಸುವ ವಕೀಲರು ವಿಷಯಕ್ಕೆ ಜವಾಬ್ದಾರರಾಗಿರುತ್ತಾರೆ.

## 🔒 Pseudonymisation Procedure

Reader ಏಜೆಂಟ್ ಡೌನ್‌ಸ್ಟ್ರೀಮ್ LLM-ಚಾಲಿತ ಏಜೆಂಟ್‌ಗಳು ವಾಸ್ತವಗಳನ್ನು ನೋಡುವ ಮೊದಲು — ಪಕ್ಷಗಳ ಹೆಸರುಗಳು, ವಿಳಾಸಗಳು, ಗುರುತಿನ ಸಂಖ್ಯೆಗಳು, ಆರ್ಥಿಕ ಅಂಕಿಗಳು — ರಚನಾತ್ಮಕ placeholder ಗಳಿಂದ ಬದಲಾಯಿಸುತ್ತದೆ. ನಿಜವಾದ ಡೇಟಾ ಎಂದಿಗೂ LLM ಗೆ ಹೋಗುವುದಿಲ್ಲ.
