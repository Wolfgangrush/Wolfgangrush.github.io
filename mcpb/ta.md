## நிறுவல்

1. எந்த ஒரு repo க்கும் செல்லுங்கள் → Releases → சமீபத்திய `.mcpb` கோப்பைப் பதிவிறக்கம் செய்க
2. Claude Desktop App திறக்கவும் → Settings (கியர் ஐகான்) → **Extensions** → Install Extension
3. பதிவிறக்கம் செய்யப்பட்ட `.mcpb` கோப்பைத் தேர்ந்தெடுக்கவும் → Open

(குறிப்பு: Settings → **Extensions** சரியான பேனல். Plugins அல்லது Connectors அல்ல.)

## ⚠️ சரிபார்ப்பு மறுப்பு

AI தவறு செய்யலாம். ஒவ்வொரு வரைவும் ஒரு **தொடக்கப் புள்ளி**. தாக்கல் செய்வதற்கு முன், ஒவ்வொரு சட்ட குறிப்பு, மேற்கோள், உண்மை-உரிமை, பதிவகம்-தொகுப்பு சரிபார்க்கவும். தாக்கல் செய்யும் வழக்கறிஞர் உள்ளடக்கத்திற்குப் பொறுப்பாவார்.

## 🔒 Pseudonymisation Procedure

Reader முகவர் கீழ்நிலை LLM-இயக்கப்பட்ட முகவர்கள் உண்மைகளைப் பார்ப்பதற்கு முன் — கட்சியின் பெயர்கள், முகவரிகள், அடையாள எண்கள், நிதி எண்கள் — கட்டமைப்பு placeholder களால் மாற்றுகிறது. உண்மையான தரவு ஒருபோதும் LLM க்கு செல்லாது.
